#  Cipher Quest

A sleek, interactive Python application that allows users to encrypt and decrypt messages using the **Caesar Cipher** algorithm. This project demonstrates the transition from a functional Command Line Interface (CLI) to a full Graphical User Interface (GUI).

---

##  Features

* **Dual Interface:** Includes both a **CLI version** for terminal fans and a **GUI version** for desktop users.
* **Secure Logic:** Handles uppercase, lowercase, and special characters seamlessly.
* **User-Friendly GUI:** Built with Python's `Tkinter` library featuring a dark-themed, modern aesthetic.
* **Error Handling:** Robust validation to prevent crashes from invalid shift values.
* **Gamified Challenges:** (CLI Version) Includes a "Codebreaker" mode to test your decryption skills.

---

##  Tech Stack

* **Language:** Python 3.x
* **Library:** Tkinter (for GUI)
* **Logic:** Caesar Cipher Algorithm
